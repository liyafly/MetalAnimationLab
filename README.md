# MetalAnimationLab

MetalAnimationLab is a cross-platform laboratory for studying Core Animation, Metal, rendering pipelines, and animation performance on Apple platforms.

The repository uses native UIKit and AppKit application shells, shared SwiftUI experiment interfaces, and reusable Swift Package modules.

## Initial experiments

- EXP-001: Layer Tree Basics
- EXP-002: Implicit vs Explicit Animation
- EXP-003: MTKView First Triangle
- EXP-004: Offscreen Rendering
- EXP-005: CAMetalLayer Manual Renderer
- EXP-006: Metal Particle Animation

## Roadmap

The first milestone focuses on Core Animation and Metal. Future milestones may add AVFoundation video frames, CVMetalTextureCache integration, realtime video effects, Vision, and Core ML.
